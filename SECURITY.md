# Security Policy

Please do not open public issues for vulnerabilities.

Report security concerns through GitHub private vulnerability reporting if it is
available on this repository. If it is not available, contact the maintainer
through the repository owner profile.

Include:

- What is affected.
- How to reproduce it.
- Any known impact.
- Whether credentials, generated media, or local files may be exposed.

This project is intended to run locally on `127.0.0.1`. Do not deploy the
dashboard or API publicly without adding authentication and reviewing the
storage paths.
